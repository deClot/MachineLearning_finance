Набор данных представляет собой признаковое описание компаний за различные периоды времени. Каждая строка набора данных - это набор финансовых показателей компании, агрегированных за определённый период времени. В поле правильных ответов находится число, показывающее финансовое благополучие компании на момент окончания данного периода.
Весь набор данных разделён на два поднабора, предназначенных для обучения и тестирования - файлы finance_train.csv и finance_test.csv соответственно.
Каждый из поднаборов представляет собой таблицу с признаковым описанием финансовых показателей компании и столбцом правильных ответов - число, показывающее финансовое благополучие компании.
Имеются следующие столбцы:
 - Company - идентификатор компании;
 - Time - номер периода времени;
 - Financial Distress - число - показатель финансового благополучия компании на окончание периода времени Time. Если это число меньше -0.50, то компания рассматривается как потерпевшая финансовый крах, иначе компания является финансово благополучной;
 - столбцы x1-x83 - различные финансовые показатели компании - все числовые значения, кроме столбца x80 - он содержит категориальные значения.

Задача по работе с данными: как можно точнее определять финансовое благополучие компании не тестовой выборке (finance_test.csv).

Примечания по работе с данными:
 - целевое поле представлено числом, а не классом. Данную задачу можно рассматривать как задачу регрессии, так и как задачу классификации (выполнить преобразование числового поля правильных ответов в бинарное поле финансового благополучия);
 - каждая компания в наборе данных представлена за несколько временных периодов (от 1 до 14), можно придумать способ обработки этого;
 - классы в выборке являются несбалансированными.
